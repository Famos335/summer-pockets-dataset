# summer-pockets-dataset
Корпус текстов Summer Pockets Reflection Blue, собранный через Hook и OCR. Пройдены все руты, кроме Уми, а также Ислапонг и Исмамонбатл. Есть оригинал и машинный перевод. Скрытые диалоги отсутствуют — найдены не все. Для исследований MT и анализа. Неофициально, non-commercial.


# Summer Pockets Reflection Blue — мульти-источник текстовый корпус

Коллекция текстовых баз данных визуальной новеллы **Summer Pockets Reflection Blue**, собранных разными методами (Hook + OCR). Включает оригинальный текст и машинный перевод. Корпус предназначен для исследований машинного перевода, анализа качества MT и учебных NLP-проектов и перевода игры кто готов.

---

## 📊 Что внутри

| Компонент | Описание |
|-----------|----------|
| Hook-база | Извлечение через текстовый хук (Textractor / ITH) |
| OCR-база | Распознавание с экрана для текстов, не ловящихся хуком |
| Оригинал | Текст из игры (английский) |
| Машинный перевод | Перевод (модель: DeepL / GPT / Google) |

## ⚠️ Ограничения

- **Рута Уми отсутствует** — не была пройдена в момент сбора.
- **Некоторые скрытые диалоги отсутствуют** — не удалось найти/извлечь при обычном прохождении.
- **Артефакты OCR возможны** — особенно на нестандартных шрифтах.
- **Машинный перевод не вычитан** — может содержать ошибки
