## О проекте

Этот репозиторий содержит корпус текстов визуальной новеллы **Summer Pockets Reflection Blue**. Данные собраны несколькими способами: часть текста извлечена через текстовый Hook, часть восстановлена через OCR для сцен, которые не удалось стабильно получить хуком.

В корпус входят оригинальные строки и машинный перевод. Материалы можно использовать для экспериментов с MT, оценки качества перевода, лингвистического анализа и подготовки фанатских или учебных инструментов.

## Быстрый старт

Данные доступны в двух видах:

- [`translation_record/`](translation_record/) — распакованная папка с SQLite-базами;
- [`translation_record.7z`](translation_record.7z) — компактный архив той же папки.

Главная база корпуса:

```text
translation_record/SummerPocketsRB_1771957083.197056_c0538fba-7b9a-4ac0-b0a6-511637015b17.sqlite
```

Техническое описание структуры и примеры команд находятся в [`DATASET.md`](DATASET.md).

## Что внутри

| Компонент | Описание |
| --- | --- |
| Основная SQLite-база | 58 348 строк в таблице `artificialtrans` |
| OCR-база | 8 016 строк, распознанных с экрана для сцен, которые не ловились хуком |
| Кеши переводчиков | SQLite-кеши ChatGPT-compatible, DeepL, Google, Yandex, Papago и Translate.com |
| Архив | `translation_record.7z` для скачивания всех данных одним файлом |
| Скрипт статистики | `translation_record/script.py` для быстрой проверки основной базы |

## Ограничения

- **Рута Уми отсутствует**: на момент сбора она не была пройдена.
- **Некоторые скрытые диалоги могут отсутствовать**: не все варианты удалось найти или извлечь при обычном прохождении.
- **Возможны OCR-артефакты**: особенно в строках с нестандартными шрифтами, эффектами или сложным фоном.
- **Машинный перевод не вычитан**: текст может содержать смысловые, стилистические и терминологические ошибки.

## Для чего можно использовать

- исследование и сравнение систем машинного перевода;
- анализ качества автоматического перевода визуальных новелл;
- подготовка учебных NLP-датасетов;
- поиск терминологии, повторяющихся реплик и сценарных паттернов;
- черновая база для дальнейшей ручной вычитки.

## Быстрая проверка

```bash
python translation_record/script.py
```

Скрипт выведет количество строк, общий объем символов и базовую статистику по колонке `source` в основной базе.
